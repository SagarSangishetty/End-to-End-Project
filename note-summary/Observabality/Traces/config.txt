Acts as the middle layer between the application and Jaeger.

Flow:

Application
      │
      ▼
OTEL Collector
      │
      ▼
Jaeger
Important Configuration
fullnameOverride
fullnameOverride: otel-collector

This is important because the application sends traces to:

otel-collector:4317

The Kubernetes Service name must match the endpoint configured in the application.

Otherwise the application cannot send traces.

OTEL Collector Exporter

Configured the collector to export traces to Jaeger.

exporters = {
  "otlp/jaeger" = {
    endpoint = "jaeger:4317"
    tls = {
      insecure = true
    }
  }
}

Meaning:

OTEL Collector
      │
      ▼
Jaeger Collector (4317)
Application Configuration

Application configured with

OTEL_EXPORTER_OTLP_ENDPOINT=otel-collector:4317

Application exports traces to the OTEL Collector instead of directly to Jaeger.
